== Description ==

Plugin have functionality to export post in one click to csv format.

== Installation ==

It has one click installation no need any configrations.

e.g.

1. Upload `posts-export.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
